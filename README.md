# 自动关机助手

一个简单易用的 Windows 自动关机工具，支持定时关机、倒计时提醒和配置保存功能。

## 功能特点

- 📅 支持设置定时关机时间
- 🔄 自动保存配置，下次启动自动恢复
- 🔔 关机前声音提醒
- 💻 系统托盘图标操作
- ⏰ 实时显示剩余时间
- ❌ 随时可以取消关机计划

## 使用方法
直接在右侧 release 下载exe使用

1. 在系统托盘找到关机图标
2. 右键点击图标可以：
   - 设置关机时间（预设时间或自定义）
   - 查看剩余时间
   - 取消关机计划
   - 退出程序

## 注意事项

- 关机前20秒会开始倒计时（通知弹窗+声音）
- 倒计时期间会有声音提醒（每10秒一次）
- 可以随时通过右键菜单取消关机
- 设置会自动保存在 `shutdown_config.ini` 文件中

## 配置文件说明

配置文件 `shutdown_config.ini` 会自动创建在脚本所在目录，包含：
- 关机时间设置
- 关机计划是否开启

## 系统要求
- Windows 操作系统
- AutoHotkey v1.1 或更高版本

## 快捷键

- 右键点击托盘图标访问所有功能
- 菜单项支持键盘快捷键：
  - T：打开时间设置菜单
  - C：取消关机
  - X：退出程序
